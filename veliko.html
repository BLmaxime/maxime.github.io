Présentation du TP - Veliko

Aperçu du Projet

  • Développement d'une application permettant de vérifier la disponibilité des vélos Vélib’ en temps réel.
  • Affichage interactif des stations avec le nombre de vélos disponibles et d’espaces libres.
  • Utilisation de Symfony pour la gestion backend et des API pour récupérer les données des stations.

Informations utiles

  • Langages : TWIG, PHP, JavaScript
  • Framework : Symfony
  • Base de données : MySQL
  • API : API Vélib

Difficultés Rencontrées

  • Intégration des données Vélib.
  • Affichage dynamique des stations sur la carte interactive.
  • Gestion des migrations avec Doctrine.
  • Gestions des resérvations de vélo avec la modification dans la bdd et l'api.

Compétences Acquises

  • Maîtrise de Symfony et gestion des bases de données avec Doctrine.
  • Utilisation des API externes pour afficher des données en temps réel.
  • Intégration de cartes interactives.

Conclusion

  • Expérience enrichissante en développement web avec des technologies modernes.
  • Renforcement des compétences en gestion des API et traitement des données en temps réel.
  • Importance de l'optimisation des performances et de la gestion des erreurs.
  • Préparation pour des projets web plus complexes et évolutifs.
